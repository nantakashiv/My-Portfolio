ATM Cash Prediction System

🏢 PT Collega Inti Pertama (Telkom Group)
📅 September 2023 – February 2025
🎯 Data Scientist Intern
Python Graph Attention Network GNN Deep Learning Predictive Modeling Machine Learning Data Analysis Jupyter Notebook

Project Overview

This project was developed during my internship at PT Collega Inti Pertama, a subsidiary of Telkom Group. The core objective was to build an intelligent model capable of predicting ATM cash needs based on historical transaction patterns, geographic relationships, and temporal trends across the ATM network.

Rather than using conventional time-series models, I implemented a Graph Attention Network (GAT) — a type of Graph Neural Network — to capture both the spatial relationships between ATMs (e.g., nearby ATMs may share similar demand patterns) and temporal transaction trends. This approach allowed the model to make highly context-aware predictions that significantly outperformed baseline methods.

Key Achievements

  • Developed an advanced GAT model to analyze spatial and temporal relationships between ATMs, capturing transaction trends and geographic dependencies across the ATM network.
  • Achieved a model accuracy of 92.38% with strong precision, recall, and F1-score metrics, ensuring highly reliable cash need predictions.
  • Provided actionable insights on cash stock levels and estimated days until depletion, directly supporting ATM management teams in optimizing replenishment schedules.
  • Led data-driven initiatives to reduce the risk of cash shortages and ensure ATM service continuity, improving overall operational efficiency.
  • Managed the full data science pipeline: data collection, preprocessing, feature engineering, model training, evaluation, and insight reporting.

Project Overview

E-Not is an enterprise-grade legal management SaaS platform designed from the ground up for Notary & PPAT office operations. As Frontend Developer and System Analyst, I was responsible for both the business analysis and the complete frontend engineering of the system — translating complex legal workflows into intuitive, structured digital modules.

The project required deep understanding of notary office operations, careful system design, and a strong technical execution using modern frontend technologies. My role spanned from drafting the technical blueprint to building the fully functional multi-module frontend.

System Modules Built

  • Case Management — End-to-end tracking of notarial cases from intake to closing, with status workflows and client assignment.
  • Legal Workflow Tracking — Multi-step workflow management with stage-based progression, validation, and handoff notifications.
  • Document Management & Generator — Dynamic document generation, PDF preview, and structured document lifecycle management.
  • Payment Monitoring — Invoice tracking, payment status monitoring, and financial reporting for legal services.
  • Client Repository — Centralized client database with case history, contact information, and document records.
  • Audit Trail Dashboard — Full activity logging and audit trail visualization for compliance and accountability.
  • Role-Based Interface — Dynamic UI rendering and access control based on user roles (admin, notary, staff, client).
  • Reminder System — Automated reminder scheduling for document deadlines, appointments, and legal obligations.

System Analysis Deliverables

In addition to building the frontend, I produced a full technical proposal and system blueprint that included feature roadmap planning, module architecture structure, database design concepts, user role definitions, system workflows, and SaaS deployment readiness assessment. This documentation served as the foundation for the full system development plan.

Technical Approach

Built with enterprise application principles at the core — the frontend architecture prioritizes scalability, maintainability, UI consistency, and optimal user experience. The component structure follows a modular, reusable pattern using Next.js App Router, React with TypeScript for type safety, and Tailwind CSS for a consistent, responsive design system.
